This site documents transit-map, a webapp for visualizing transit data from GTFS and its HTTP API. The source code for transit-map (and this website) can be found on GitHub. An instance of the transit-map website is currently available at j-markiewicz.github.io/transit-map.

API

The transit-map API makes transit data available and the sources for this data configurable. Data is available without authentication, while configuration requires an API token to edit. More detailed documentation about the API and an interactive testing tool is available on its own page.

An overview of data types used in the API is available separately.

Website

The transit-map website is available at j-markiewicz.github.io/transit-map (and it'll probably work until the free trial for the VPS hosting the API server runs out in late December 2024). An overview of the features of the website is available here.

All transit data available via the transit-map API is based on GTFS Schedule and GTFS Realtime data. Here's some information about GTFS.
